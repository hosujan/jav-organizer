# Web UI Guide

## Organize
- Choose local directory.
- Start processing queue.
- View progress, logs, and history.
- Selected directory is persisted for next launch.

## Browse
- Featured hero and recommendation rails.
- Discovery filters for quick narrowing.
- Button to dedicated `All Titles` page.

## All Titles
- Full catalog search.
- Multi-facet filtering (year, genre, actress, studio, duration, rating).
- Sorting controls and responsive card grid.

## Video
- Metadata detail page.
- Preview area and metadata facts.
- Action to open full playback.

## Watch
- Full local video playback.
- Resume/start-over from saved progress.
- Up Next recommendation list.
